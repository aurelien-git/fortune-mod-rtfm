# fortune-mod-security
> Because security is also a cool way to learn tips and tricks in CS
> here a fortune-mod for that

---

## How to?
git clone the stuff, then pacman -U fortune-mod-security.xz
and

Just write
```shell
command fortune security
```
To your .bashrc
